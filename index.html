Fiche de note sur l'imparfait en français


- L'imparfait est un temps passé qu'on utilise souvent dans un conte, un récit pour décrire une personne, un animal ou une chose.

- Il indique généralement une action qui a duré ou qui s'est répétée dans le passé, sans limites définies. Tous les verbes ont la même terminsaison même les auxiliaires !

Imparfait Manger Grandir Venir
Je Mangeais Grandissais Venais
Tu Mangeais Grandissais Venais
Il Mangeait Grandissait Venait
Nous Mangions Grandissions Venions
Vous Mangiez Grandissiez Veniez
Ils Mangaient Grandissaient Venaient

On le voit içi avec un verbe de chaque groupe les terminaisons restent : "ais", "ais", "ait", "ions", "iez", "aient"
